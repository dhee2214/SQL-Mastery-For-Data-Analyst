## SQL-SELECT
Select is used to retrive the data from the one or morle columns from table
Syntax:
       ``` Select Column_name
           From Table_Name; ```


## SQL-SELECT-DISTINCT
DISTINCT is used to select/return unique values from the database
Syntax:
        ``` Select DISTINCT Column_name
            From Table_name; ```

## SQL SELECT (*)
* is used to return all the cloumns from the table
Syntax:
        ``` Select *
            from Table_name; ```

## AS
AS is used to assign a temporary name (alias) to a column or table, making the output more readable.
Syntax:
        ``` SELECT column_name AS alias_name
            FROM table_name; ```

## Top
TOP is used to retrieve a specified number of rows from a table.
Syntax:
        ``` SELECT TOP (n) column_name
            FROM table_name; ```
